Sinus

Built-in signals ›› Built-in signals types ››

Generate a sinusoidal waveform.



Uses the following math properties


    • Frequency: Sinusoid frequency in Hertz.


    • Magnitude: Peak to peak signal magnitude.


    • Offset: Signal offset with respect to zero.


    • Phase: Sinusoid phase offset.
