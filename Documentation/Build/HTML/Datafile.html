Data file

Data analysis ›› Graphic window ››

The data file format used by the analysis window is CSV (Column Separated Value)


Columns must be separated by a semi-comma character [ ; ].

Values decimal separator must be point [ . ] or comma [ , ].


First line of the file contains names of data channels.


First column must be abscisse value (X axis values).
